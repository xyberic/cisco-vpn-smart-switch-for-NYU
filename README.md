# NYU/NYUSH VPN Smart Switch & Automator

🚀 本项目是一套专为纽约大学 (NYU/NYUSH) 同学打造的 **Cisco Secure Client (AnyConnect) VPN 全平台自动化智能切换工具**。

旨在解决每天上课、写代码、查文献时，频繁手动开启思科客户端、敲长密码、输入 `push` 以及应付 Duo 验证的繁琐痛点。通过本项目，你可以实现 **“无脑双击/单命令控制，未连秒连，已连秒断”** 的极致网络体验。

---

## ✨ 核心特性 (Features)

* **🔄 智能双向切换 (Smart Switch)**：运行脚本时全自动检测网络状态。若 VPN 处于断开状态，则自动启动登录隧道；若已连接，则安全切断，全天候只需交互这一个入口。
* **🛠️ 强力路径探测**：地毯式兼容了 Windows (v4.x/v5.x 交叉路径)、macOS、Linux 下思科客户端的所有默认安装路径，免去手动配置环境变量的烦恼。
* **⚡ 彻底免疫闪退与乱码**：Windows 端强制 UTF-8 内核编码，移除了所有可能导致 CMD 编译器多字节对齐错位的特殊符号，稳定运行不闪退。
* **🔒 完美处理控制字符**：原生支持并保护了密码中包含特殊脚本控制字符的教工/学生账户，确保凭据精准递交。

## 📥 下载与部署指南 (Installation & Usage)

为了保持代码仓库的整洁，所有的功能脚本均已打包并发布在 [GitHub Releases](../../releases) 页面中。请前往 Release 页面下载对应系统的脚本文件。

### 1. Windows 用户使用指南 (`.bat` 脚本)

#### ⚙️ 第一步：基础配置
1. 前往本仓库的 **Releases** 页面，下载最新的 `NYU_VPN.bat` 脚本文件。
2. 右键点击下载好的 `NYU_VPN.bat`，选择 **编辑**（或用记事本打开）。
3. 在脚本中找到对应位置，将占位符替换为你自己的真实校园网凭据：
   * 将 `你的NetID账号`修改为你自己的 NetID。
   * 将 `你的第一密码` 修改为你自己的 NYU 密码。
   * *注意：如果你的密码末尾带有 `&` 符号，请务必保留其前面的 `^` 转义符（例如：`MyPassword^&`）。*
4. 点击记事本左上角 **文件 ➔ 另存为**，确保窗口最下方的 **编码 (Encoding)** 选择为 **`UTF-8`**，然后保存覆盖。

#### 🎯 第二步：日常运行
* **一键开关**：直接双击运行 `NYU_VPN.bat`。
  * **若处于未连接状态**：脚本会强行清理后台冲突进程，秒发验证请求。当你看到黑窗口提示 `Please approve DUO PUSH on your phone...` 时，**留意手机 Duo 弹窗并点击批准**。连接成功后，黑窗口会在 2 秒内静默自动关闭。
  * **若处于已连接状态**：再次双击，脚本会利落地切断校园网 VPN 连接，随后黑窗口自动蒸发。

### 2. macOS & Linux 用户使用指南 (`.sh` 脚本)

#### ⚙️ 第一步：下载与权限解锁
1. 前往本仓库的 **Releases** 页面，下载最新的 `nyu_vpn.sh` 脚本文件。
2. 打开 **终端 (Terminal)**，使用 `cd` 命令切换到你下载该文件的目录下（例如 `cd ~/Downloads`）。
3. 执行以下命令，为脚本赋予系统级可执行权限：
   ```bash
   chmod +x nyu_vpn.sh
📝 第二步：凭据配置
使用本地文本编辑器（如 nano 或 VS Code）打开 nyu_vpn.sh。

在代码中段的 Here-Doc (<<EOF) 区域，将占位符 your_netid 和 your_password 替换为你自己的真实凭据。

保存并关闭文件。

🚀 第三步：终极偷懒技巧（命令行宏挂载）
为了让你连 ./nyu_vpn.sh 都不用打，建议将其挂载为系统全局别名：

在终端中根据你的 Shell 类型打开配置文件（Mac 默认 Zsh 执行 nano ~/.zshrc，Linux 默认 Bash 执行 nano ~/.bashrc）。

滚动到文件最底部，添加一行（假设你将脚本放在了家目录下）：

Bash
alias vpn="~/nyu_vpn.sh"
保存退出后，执行 source ~/.zshrc（或 source ~/.bashrc）使其立刻生效。

💡 日常使用体验：此后你随时随地打开终端，只需敲击 vpn 三个字母并回车，即可实现全自动的网络连接触发或断开，并在 1 秒后自动干净地退出终端会话。
